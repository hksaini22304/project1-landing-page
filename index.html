Lab 1: CSS Layout w/ Flexbox and Grid

Due: September 4, 2025

In this lab, I looked at current CSS layout approaches like Flexbox and Grid to develop responsive, well-structured web designs. The emphasis was on knowing how to efficiently align, distribute, and arrange content across many screen sizes. By using these layout approaches, I was able to practice creating adaptable page layouts that maintained visual harmony, flexibility, and clean design, all of which are necessary abilities for professional web development.

Lab

Lab 2: Transitions & Animations

Due: September 11, 2025

For this lab, I created a landing page for an imitation cosmetics company with many major components, including a hero banner, product mockups, an about section, a contact form, and an email newsletter submission. To bring the site to life, I used CSS transitions and animations, as well as a fluid scrolling brand display to replicate press and blog content. This project enabled me to blend layout design with interactive motion, resulting in a polished landing page that looks like a professional company website.
